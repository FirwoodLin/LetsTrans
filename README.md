# LetsTrans

## ⭐项目简介

LetsTrans 是一个 Web 端计算机辅助翻译系统，旨在为用户提供高效、便捷的翻译文本管理服务。

# 🚀技术选型

## 前端
vue3 + vite + pinia + ESlint 

## 后端

语言以及框架：Golang + Gin

数据库：PostgreSQL






## 📕 项目文档

见 [docs](docs) 目录。
